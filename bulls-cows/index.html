Bulls and Cows

How to play this game?

Computer generated a number base on the difficult level.

  • easy - 4 digit (default)
  • hard - 7 digit

2 players try to guess the correct number base on feedback from the system. Both won't know other guess, only the feedback show out.

Each correct digit and at the right position called (bull), each correct digit but at wrong position called (cow).

The first player guess correctly the number win. Good luck!
